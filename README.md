## Motivo de la elección



Conocimiento del lenguaje: trabaja con Python, un lenguaje que conozco y cuya sintaxis es clara, lo que facilita aprender el framework y desarrollar el proyecto.
SEO: Django permite generar páginas desde el servidor mediante plantillas HTML, por lo que el contenido puede ser leído fácilmente por los motores de búsqueda.
Tiempo disponible: incluye herramientas listas para usar, como manejo de rutas, conexión con bases de datos, autenticación, panel administrativo y seguridad. Esto permite avanzar más rápido sin construir todo desde cero.
Arquitectura del proyecto: para este laboratorio no necesito separar frontend y backend. Django permite manejar la lógica, las vistas, las rutas y la base de datos dentro de un mismo proyecto, reduciendo la cantidad de tecnologías que debo configurar e integrar.
